## 🚀 如何使用

### 安装（首次使用）

**Windows 用户**:
```cmd
install.bat
```

**Linux/macOS 用户**:
```bash
chmod +x install.sh
./install.sh
```

### 使用工具

安装完成后，重启终端，然后可以在**任何目录**使用：

```bash
# 交互模式（最简单）
vscodegen

# 查看帮助
vscodegen --help

# 查看版本
vscodegen --version

# 指定路径生成
vscodegen --path ./src --output ./

# 包含特定项目
vscodegen --include "MyProject,WebApi"

# 排除测试项目
vscodegen --exclude "*.Tests"
```

## 📁 生成的文件

工具会在指定目录创建 `.vscode` 文件夹，包含：
- `launch.json` - VSCode 调试配置
- `tasks.json` - 构建任务配置
